开发工具

GithubPages+hexo博客的搭建：
本文主要介绍的的是使用github pages搭建博客的完整过程，博主使用的是hexo框架，主题使用的是blackblue,同时还会介绍几个小问题的解决过程。

使用Github Pages

博主以前曾搭建过个人博客，当时选择的方案是vps+独立域名+worldpress,比较可惜的是写了几篇博客之后就废弃了，主要是比较迷茫，不知道写啥，好吧，说到底还是懒。过了一年后，vps和域名都到期了，也就不了了之。现在呢，用github pages搭建个人博客就不会有vps到期的问题了，只要github不倒闭，那么你的博客还是会一直存在的。使用github pages写博客相对于在csdn和简书上更加灵活，相对于vps来说更加简便。
首先你得有一个github账号，然后新建一个仓库：你的用户名.github.io,之后你就可以选择不同的框架来构建你的博客，前几年比较流行的是 Jekyll，如今hexo这个更加简便的框架逐渐取代了它的地位，下面主要介绍hexo的使用。

使用主题出现的一些问题

  1. 评论系统
    这里要说的是black-blue这个主题中提供了三种评论系统给开发者选择，分别是畅言（搜狐的），友言（开源的）和gitment(github的)，最后还有disqus。首先畅言这个东西注册后使用需要网站进行icp备案，过程比较漫长，所以pass；友言这个东西目前来看好像已经挂了，所以也pass，Disqus是国外的一个公司的产品，登录方式时gmail，facebook等方式，对于大陆人很不友好，最后只剩下gitment了，gitment是github提供的一个评论系统，用户使用github登录然后才能评论，有时候确实有些不方便，不过还凑合着用吧。
    使用gitment首先要注册 OAuth Application,其中callbackURL部分一定要填写你的博客地址，之后会得到一个 client ID 和一个 client secret，用于之后的用户登录。
